# Turbo Mapper

[![License](https://img.shields.io/github/license/kForth/Turbo-Mapper)](https://github.com/kForth/Turbo-Mapper/blob/main/LICENSE)

A tool to easily plot demand lines on turbocharger compressor maps and find the right turbo for your application.
